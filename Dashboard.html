Oct 6, 2025 | DocuWrite Pro 3.2 | Migration Update

DocuWrite Pro has been officially migrated from Google Sites to GitHub! This move ensures faster performance, full control over the source code, and a stronger, more flexible foundation for future innovations. The entire system remains 100% client-side, preserving privacy while unlocking new possibilities such as cookies, browser permissions, and advanced file handling.

- Mohtadi Azad Mahi, Creator of DocuWrite Pro

What's New in 3.2?

  • Migration from Google Sites to GitHub Pages.
  • Improved page loading speed and caching efficiency.
  • Enhanced local data storage using cookies and permissions.
  • Optimized internal linking and file structure for better performance.
  • Minor visual improvements and overall stability updates.
